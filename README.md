# 🩺 Caderneta Digital de Estágio - FMC

Sistema web desenvolvido para automatizar e organizar o registro de presença e atividades de alunos em campos de estágio hospitalar da **Faculdade de Medicina de Campos**.

## 🚀 Link do Projeto
Acesse o sistema em produção: 
👉 [https://alexandro391.github.io/caderneta-digital-estagio/](https://alexandro391.github.io/caderneta-digital-estagio/)

## 🛠️ Funcionalidades
- **Registro de Presença:** Interface intuitiva para alunos registrarem entrada e saída.
- **Painel do Preceptor:** Área restrita para validação de atividades.
- **Banco de Dados em Tempo Real:** Integração com Google Firebase para armazenamento seguro.
- **PWA (Progressive Web App):** Pode ser instalado no celular como um aplicativo (suporte offline).

## 🔒 Segurança e Tecnologias
- **Frontend:** HTML5, CSS3 (Design Responsivo) e JavaScript (ES6+).
- **Backend:** Google Cloud Firestore (NoSQL).
- **Segurança:** Chaves de API protegidas por restrições de referenciador HTTP e limitação de escopo de APIs.

## 📂 Estrutura do Repositório
- `index.html`: Portal principal de acesso.
- `preceptor.html`: Sistema de controle para supervisores.
- `manifest.json` / `sw.js`: Configurações para funcionamento como Aplicativo (PWA).
